# linux
## command
1. cd (change directory)
- cd ..  ->   parent directory
- cd /   ->   root
- cd ~   ->   home 
- cd -   ->   previous
***
2. ls (list)
- ls -l   ->  show detiles
- ls -a   ->  show hidden files
***
3. cat (concatenate file and print output)
- cat -v   ->  show hidden character
***
4. chmod (change read,write,execute for each group)
- chmod +x  ->  user can execute
***
5. grep
***
6. run python/bash
