# move
- h -> left
- j -> down
- k -> up
- l -> right
- 0 -> head of line
***
# exit
- :q! -> discard any change
- :wq -> save changes
***
# edit
- i -> insert front of cursor
- a -> append end of line
# delete
- x (in normal mode)->delete character on cursor
- dw -> delete a word
- d2w -> delete two words
- d$ -> delete behind cursor
- dd -> delete a line
# undo
- u -> back a step
- U -> back a series of steps (usually a line)
